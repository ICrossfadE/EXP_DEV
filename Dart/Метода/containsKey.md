Метод `containsKey` в Dart використовується для перевірки наявності певного ключа в колекціях типу `Map`. Він повертає значення `true`, якщо зазначений ключ існує у мапі, і `false` — якщо ні.

### Як це працює:

- `containsKey` використовується виключно з об'єктами типу `Map`.
- Він перевіряє наявність конкретного ключа в мапі без необхідності звертатися до його значення.

```dart
Map<String, int> ages = {
  'Alice': 25,
  'Bob': 30,
  'Charlie': 35,
};

// Перевіряємо, чи є ключ 'Alice' в мапі
bool hasAlice = ages.containsKey('Alice');
print(hasAlice); // Виведе: true

// Перевіряємо, чи є ключ 'David' в мапі
bool hasDavid = ages.containsKey('David');
print(hasDavid); // Виведе: false
```
### Використання у логіці:

Метод `containsKey` часто використовується для того, щоб уникнути звертання до неіснуючого ключа в мапі та виключення помилок. Це дозволяє перевірити, чи можна безпечно звертатися до значення за вказаним ключем.

```dart
if (ages.containsKey('Bob')) {
  print('Bob is ${ages['Bob']} years old.');
} else {
  print('Bob is not in the list.');
}
```

Перед тим як звернутися до значення `ages['Bob']`, спершу перевіряємо, чи існує цей ключ у мапі, щоб уникнути помилки доступу до неіснуючого ключа.

### Висновок:

- **`containsKey`** — це простий і швидкий спосіб перевірити, чи існує конкретний ключ у мапі.
- Він корисний для уникнення помилок при доступі до неіснуючих ключів.
- Метод працює для будь-яких типів ключів, що використовуються в `Map`, навіть якщо значення для цього ключа — `null`.
